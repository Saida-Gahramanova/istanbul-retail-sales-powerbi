# Istanbul Retail Sales Dashboard (Power BI)

## Overview
This is my first Power BI portfolio project where I analyzed retail sales data from shopping malls in Istanbul.
The goal of this project was to understand overall sales performance, customer behavior, and product trends by building an interactive dashboard.

## Dataset
* Source: Kaggle
* 503 transaction records
* Includes customer info, product categories, payment methods, and shopping mall locations

## Data Preparation
Even though the dataset was mostly clean, I still performed several preparation steps in Power Query:
* Checked and corrected data types
* Removed duplicate rows
* Created a new column: "Total Sales (quantity × price)"
* Extracted date fields (year, month) for time analysis
* Created "Age Groups" to better analyze customer segments


## Dashboard Pages
### Sales Overview
Shows high-level KPIs such as total revenue, transactions, units sold, and average order value, along with sales trends over time.

### Customer Insights
Analyzes customer distribution by age group and gender, and compares spending behavior.

### Product Analysis
Focuses on category performance using revenue and quantity metrics.

### Sales Channels & Locations
Explores performance across shopping malls and payment methods.


## Tools Used
* Power BI
* Power Query
* DAX
